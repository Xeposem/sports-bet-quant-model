  <action>
    Create `src/backtest/reporting.py` with:

    **ROI breakdowns** — `compute_roi_breakdowns(conn, model_version="logistic_v1") -> dict`:
    - Query backtest_results with GROUP BY for each dimension.
    - Return dict keyed by dimension name, each value is a list of dicts:
      ```python
      {
          "by_surface": [{"label": "Hard", "n_bets": 450, "kelly_roi": 0.032, "flat_roi": 0.015, "total_pnl_kelly": 14.4, "total_pnl_flat": 6.75, "low_confidence": False}, ...],
          "by_tourney_level": [...],
          "by_year": [...],
          "by_ev_bucket": [...],   # >0%, >2%, >5%, >10%
          "by_rank_tier": [...],   # top10, top50, top100, outside100
          "overall": {"n_bets": ..., "kelly_roi": ..., "flat_roi": ..., ...},
      }
      ```
    - EV bucket logic: CASE WHEN ev >= 0.10 THEN '>10%' WHEN ev >= 0.05 THEN '>5%' WHEN ev >= 0.02 THEN '>2%' ELSE '>0%' END.
    - Rank tier logic: Use the bet-on player's rank. If betting on winner, use winner_rank; if betting on loser, use loser_rank. Group into top10 (<= 10), top50 (11-50), top100 (51-100), outside100 (>100 or NULL).
    - low_confidence flag: True when n_bets < 30. Always display n_bets alongside ROI.
    - Kelly ROI = SUM(pnl_kelly) / SUM(kelly_bet) WHERE kelly_bet > 0. Flat ROI = SUM(pnl_flat) / SUM(flat_bet).

    **Calibration plots** — `generate_calibration_plots(conn, model_version, output_dir) -> list[str]`:
    - Use `matplotlib.use("Agg")` at top of module (before pyplot import).
    - For each fold_year in backtest_results: gather calibrated_prob and outcome, call calibration_curve_data(outcomes, probs, n_bins=10).
    - Plot per-fold calibration: diagonal reference line + data points. Save as `calibration_{fold_year}.png`.
    - Aggregate all folds: calibration_curve_data on combined data. Save as `calibration_aggregate.png`.
    - Call `plt.close(fig)` after each savefig to prevent memory leak.
    - Return list of created file paths.

    **Calibration data storage** — `store_calibration_data(conn, y_true, y_prob, fold_label, model_version)`:
    - Call calibration_curve_data, store results as JSON arrays in calibration_data table via INSERT OR REPLACE.

    **Bankroll curve** — `generate_bankroll_curve(conn, model_version, output_dir) -> str`:
    - Query backtest_results ordered by tourney_date, id (chronological bet order).
    - Plot bankroll_after as a line chart over time. X-axis = tourney_date, Y-axis = bankroll.
    - Add horizontal line at initial bankroll ($1,000) for reference.
    - Save as `bankroll_curve.png`. Return file path.

    **Print summary** — `print_summary(summary, breakdowns)`:
    - Print walk-forward summary stats (folds, bets, final bankroll).
    - Print ROI breakdown tables to stdout in readable format.
    - Flag low-confidence buckets with asterisk and "(n=XX)" annotation.

    Write RED tests first in `tests/test_backtest_reporting.py`. Tests should use in-memory SQLite with manually inserted backtest_results rows (no need for full walk-forward — unit test the reporting layer independently). Verify tests fail, then implement.
